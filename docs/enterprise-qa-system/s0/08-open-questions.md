## 2. S1 可先采用的安全默认

在 OQ 未回答时，S1 只使用 `demo_corp` 合成租户、Fake Provider、合成文档和本地开发 OIDC。不得连接生产目录、上传真实文档、使用个人 API Key 处理企业数据或承诺生产 SLA。

## 3. 回答格式

每个决定记录：`ID、选择、适用范围、决定人、日期、证据、复审触发器`。答案写入 `decision-log.md`；改变技术/安全边界时同步建立或更新 ADR。
