# ADR-029：证据门槛和安全拒答优先于模型调用

- 状态：Accepted
- 日期：2026-07-16

## 背景

让模型自行判断“有没有资料”会把无授权、低相关或攻击内容发送给 Provider，并产生幻觉和费用。

## 决策

检索层先应用最低 final relevance 与 query coverage。无候选、证据不足或直接攻击时生成固定安全回复、零模型 usage ledger 并记录 abstention reason。权限不可见和无资料对外合并语义，内部审计区分。

## 后果

减少外发、成本和误答，并可独立评测拒答；阈值可能造成过度拒答，必须在固定数据集上版本化调参。首轮评测将 coverage 从 0.20 提升到 0.34。
