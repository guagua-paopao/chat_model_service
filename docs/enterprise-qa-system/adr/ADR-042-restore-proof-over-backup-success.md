# ADR-042：恢复成功证据优先于备份任务成功状态

- 状态：Accepted
- 日期：2026-07-16
- Owner：SRE / DBA / Data Owner

## 决策

- Gate 必须验证恢复后的数据库可读、身份仍可解析、关键业务记录存在、对象哈希一致和服务 readiness。
- 本地演练使用 SQLite 在线备份 API 和对象目录副本，在隔离目录中恢复，不破坏开发数据。
- 生产必须分别验证 PostgreSQL PITR、对象版本恢复、配置/密钥重新注入、DNS/Ingress 切换和审计链连续性。
- 目标 RPO 15 分钟、RTO 60 分钟是待 Owner 签字的基线；本地毫秒级结果不能外推。

## 后果

“备份完成”不再被视为可恢复证明；生产 No-Go 直到隔离恢复演练和正式签字完成。
